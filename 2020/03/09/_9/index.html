# CSS居中布局

在日常开发过程中，经常会遇到各种居中的布局需要，因此特别汇总一下水平和垂直居中的CSS样式方法。

# 水平居中

首先先来看看水平居中的布局方法，假定父子元素均不定宽度。

<div class="parent">
  <div class="child">DEMO</div>
</div>
  • 1、父text-align+子inline-block
.parent{
  text-align: center;
}
.child{
  diplay:inline-block;
}

特别注意: text-align属性会继承，子元素如果不需要则需要补充text-align进行覆盖。

  • 2、子absolute+transform
.parent{
  position:relative;
}
.child{
  position:absolute;
  left:50%;
  transform:translateX(-50%);
}
  • 3、父flex
.parent{
  display:flex;
  justify-content:center;
}
  • 4、子table+margin
.child{
  display:table;
  margin: 0 auto;
}

以上四种方法皆能完成水平居中布局。

# 垂直居中

垂直居中有三种布局方法，同样假定父子元素高度均不定。html代码如下：

<div class="parent">
  <div class="child">DEMO</div>
</div>
  • 1、父table-cell+vertical-align
.parent{
  display:table;
  vertical-align:center;
}
  • 2、子absolute+transform
.parent{
  position:relative;
}
.child{
  position:absolute;
  top:50%;
  transform:translateY(-50%);
}
  • 3、父flex
.parent{
  display:flex;
  align-items:center;
}

# 水平垂直居中

讲完水平和垂直居中的布局方法，下面来说一下水平且垂直居中的布局方法，如大家所料将上面水平和锤子方法合并在一起就可以完成水平垂直布局，所以方法有以下三种：

  • 1、水平1+垂直1
.parent{
  display:table;
  vertical-align:center;
  text-align:center;
}
.child{
  display:inline-block;
}
  • 2、水平2+垂直2
.parent{
  position:relative;
}
.child{
  position:absolute;
  left:50%;
  top:50%;
  transform:translate(-50%,-50&);
}
  • 3、水平3+垂直3
.parent{
  display:flex;
  justify-content:center;
  align-items:center;
}

综上所述，水平居中方法有4中，垂直居中方法有3种，而水平垂直居中方法有3种，分别是水平和垂直居中的三种方法按顺序对于合并在一起。
