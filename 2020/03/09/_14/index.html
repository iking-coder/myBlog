# CSS多列布局

在平常使用软件或者App应用的时候我们经常会看到一些类似新闻列表、歌单列表等的布局，所以今天我整理了一下CSS多列布局的方法，希望能对大家在开发这类界面的时候有所帮助。

# 定宽+自适应

首先我们来看第一种，定宽+自适应是用的非常多的一个布局方式，一般定宽部分都显示的是头像、图标、图片等，而自适应部分通常是标题，内容说明之类的。这种布局的html代码如下所示：


  <div class="parent">
    <div class="left">
      <p>Left</p>
    </div>
    <div class="right">
      <p>right1</p>
      <p>right2</p>
    </div>
  </div>

假设我们的定宽部分是100px；而定宽和自适应部分的间距是20px;那要完成这种布局有以下4种方法：

  • 1、float
.left{
  float: left;
  width: 100px;
}
.right: {
  margin-left: 120px;
}
  • 2、float+overflow
.left{
  float: left;
  width: 100px;
  margin-right: 20px;
}
.right{
  overflow: hidden;
}
  • 3、table+table-cell
.parent{
  display: table;
  width: 100%;
  table-layout: fixed;
}
.left,.right{
  display: table-cell;
}
.left{
  width: 100px;
  padding-right: 20px;
}
  • 4、flex
.parent{
  display: flex;
}
.left{
  width: 100px;
  margin-right: 20px;
}
.right{
  flex: 1;
}

如果两列定宽+自适应则在中间一列设置跟第一列同样的样式即可，依此了推。

# 不定宽+自适应

不定宽加自适应的html代码通上；假设不定宽部分和自适应部分间距仍为20px；要完成这种布局有以下3种方法：

  • 1、float
.left{
  float: left;
  margin-right: 20px;
}
  • 2、table
.parent{
  display: table;
  width: 100%;
}
.left,.right{
  display: table-cell;
}
.left {
  width: 1px;
  border-right: 20px solid transparent;
  background-clip: padding-box;
}
  • 3、flex
.parent{
  display: flex;
}
.left{
  margin-right: 20px;
}
.right{
  flex: 1;
}

多列不定款的情况也是以此类推。

# 定宽

在实际开发过程中，我们还会遇到那种定宽的情况，比如做一个电子商城网站，我们一行比较宽，我们会设计成显示四个商品，而商品与商品之间有一定间距，假定该间距为20px;这种情况我们的html结构如下所示：

  <div class="parent">
    <div class="column">
      <p>DEMO0</p>
    </div>
    <div class="column">
      <p>DEMO1</p>
    </div>
    <div class="column">
      <p>DEMO2</p>
    </div>
    <div class="column">
      <p>DEMO3</p>
    </div>
  </div>

.parent{
  margin-left: -20px;
}
.column{
  width: 25%;
  float: left;
  padding-left: 20px;
}

像以上这种还属于知道几列的情况，而在实际开发过程中也存在一些不知道一行不定列的情况，那么这样的情况又应该怎么做？

<div class="parent-fix">
  <div class="parent">
    <div class="column">
      <p>DEMO0</p>
    </div>
    <div class="column">
      <p>DEMO1</p>
    </div>
    <div class="column">
      <p>DEMO2</p>
    </div>
    <div class="column">
      <p>DEMO3</p>
    </div>
  </div>
</div>

.parent-fix{
  margin-left: -20px;
}
.parent{
  display: table;
  width: 100%;
  table-layout: fixed;
}
.column{
  display: table-cell;
}

像以上这种通过table方式完成布局是一种方式，另外还有一种很好的布局方式，我们来看一下：

<div class="parent">
  <div class="column">DEMO</div>
  <div class="column">DEMO</div>
  <div class="column">DEMO</div>
</div>

.parent{
  display: flex;
}
.column{
  flex: 1;
}
.column+.column{
  margin-left: 20px;
}

以上就是各种布局方法，希望能帮到大家，同时也欢迎大家指教。
