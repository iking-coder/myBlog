# JavaScript数组方法大全

数组是JavaScipt语言中一种常见的数据类型。在实际开发中，被使用到的频率非常高，它主要包含的方法有：join、push、unshift、shift、pop、slice、splice、reverse、sort、concat、reduce、some、every、filter、map、forEach等。针对数组方法整理的文章有很多，本文主要是以数组方法返回值的角度来进行整理的，一种是没有返回值的方法；一种是有返回值的方法。详见如下：

# 无返回值

首先我们来看一下无返回值的方法都有：forEach，接下来我们来逐一看一个各个方法的使用样例:

# forEach

forEach方法是用于调用数组的每个元素，并将元素传递给回调函数，对于空数组是不会执行该方法的。

  var arr = [1,2,3]
  console.log(arr.forEach(x => x + 2)) //undefined
  console.log(arr)//[1,2,3]

# 结论：forEach方法无返回值，并且不影响原数组。

# 有返回值

其次，我们来看一下有返回值的方法都有：join、，接下来我们还是来看一下各个方法的样例及其返回值分别是什么，并且是否影响原数组。详见如下：

# join

join(separator): 将数组的元素组起一个字符串，以separator为分隔符，省略的话则用默认用逗号为分隔符，该方法只接收一个参数：即分隔符。

  var arr = [1,2,3]
  console.log(arr.join(',')) //"1,2,3"
  console.log(arr)//[1,2,3]

# 结论：join方法返回数组与分隔符合并的字符串，并且不影响原数组。

# push

push方法是在数组结尾处向数组添加一个新的元素，并且返回新数组长度。

  var arr = [1,2,3]
  console.log(arr.push(4)) //4
  console.log(arr)//[1,2,3,4]

# 结论：push方法返回新数组的长度，会影响原数组。

# unshift

unshift方法是在开头向数组添加新元素，并返回数组的长度。

  var arr = [1,2,3]
  console.log(arr.unshift(5)) //4
  console.log(arr)//[5,1,2,3]

# 结论：unshift方法返回新数组的长度，并且会影响原数组。

# shift

shift方法是删除原数组第一项，并返回删除元素的值；如果数组为空则返回undefined。

  var arr = [1,2,3]
  console.log(arr.shift(',')) //1
  console.log(arr)//[2,3]

# 结论：shift方法返回数组的长度，并且会影响原数组。

# pop

pop方法是移除数组末尾最后一项，减少数组的length值，然后返回被移除的项。

  var arr = [1,2,3]
  console.log(arr.pop()) //3
  console.log(arr)//[1,2]

# 结论：pop方法返回新数组的长度，并且会影响原数组。

# slice

slice方法是返回从原数组中指定开始下标到结束下标之间的项组成的新数组。slice()方法可以接受一或两个参数，即要返回项的起始和结束位置。在只有一个参数的情况下， slice()方法返回从该参数指定位置开始到当前数组末尾的所有项。如果有两个参数，该方法返回起始和结束位置之间的项——但不包括结束位置的项。

  var arr = [1,2,3,4,5,6]
  console.log(arr.slice(4)) //[5,6]
  console.log(arr)//[1,2,3,4,5,6]
  console.log(arr.slice(2,4)) //[3,4]
  console.log(arr)//[1,2,3,4,5,6]

# 结论：slice方法返回被截取的新数组，并且不影响原数组。

# splice

splice()：很强大的数组方法，它有很多种用法，可以实现删除、插入和替换。

删除：可以删除任意数量的项，只需指定 2 个参数：要删除的第一项的位置和要删除的项数。例如， splice(0,2)会删除数组中的前两项。

插入：可以向指定位置插入任意数量的项，只需提供 3 个参数：起始位置、 0（要删除的项数）和要插入的项。例如，splice(2,0,4,6)会从当前数组的位置 2 开始插入4和6。 替换：可以向指定位置插入任意数量的项，且同时删除任意数量的项，只需指定 3 个参数：起始位置、要删除的项数和要插入的任意数量的项。插入的项数不必与删除的项数相等。例如，splice (2,1,4,6)会删除当前数组位置 2 的项，然后再从位置 2 开始插入4和6。

splice()方法始终都会返回一个数组，该数组中包含从原始数组中删除的项，如果没有删除任何项，则返回一个空数组。

  var arr = [1,3,5,7,9,11];
  var arrRemoved = arr.splice(0,2);
  console.log(arr); //[5, 7, 9, 11]
  console.log(arrRemoved); //[1, 3]
  var arrRemoved2 = arr.splice(2,0,4,6);
  console.log(arr); // [5, 7, 4, 6, 9, 11]
  console.log(arrRemoved2); // []
  var arrRemoved3 = arr.splice(1,1,2,4);
  console.log(arr); // [5, 2, 4, 4, 6, 9, 11]
  console.log(arrRemoved3); //[7]

# 结论：splice方法返回处理后的新数组，并且会影响原数组。

# reverse

reverse方法是调换数组元素的顺序。

  var arr = [1,2,3]
  console.log(arr.reverse()) //[3,2,1]
  console.log(arr)//[3,2,1]

# 结论：reverse方法返回被调换顺序后的数组，并且会影响原数组。

# sort

sort方法默认是按升序排列数组项——即最小的值位于最前面，最大的值排在最后面，但也可以通过传入排序函数进行排序，对于字符串排序会根据首字母转化为Unicode进行排序。

  var arr = [2,1,3]
  console.log(arr.sort()) //[1,2,3]
  console.log(arr)//[1,2,3]
  console.log(arr,sort((a,b) => b - a))//[3,2,1]
  console.log(arr)//[3,2,1]
  console.log(arr.sort(a,b) => a - b))//[1,2,3]
  console.log(arr)//[1,2,3]

# 结论：sort方法返回按照排序参数进行排序的新数组，并且会影响原数组。

# concat

concat方法是将参数添加到原数组中。这个方法会先创建当前数组一个副本，然后将接收到的参数添加到这个副本的末尾，最后返回新构建的数组。在无参数的情况下，它默认复制当前数组并返回副本。

  var arr = [1,2,3]
  var arrCopy1 = arr.concat()
  var arrCopy2 = arr.concat(4)
  console.log(arr)//[1,2,3]
  console.log(arrCopy1)//[1,2,3]
  console.log(arrCopy2)//[1,2,3,4]

# 结论：concat方法返回新数组，并且不会影响原数组。

# reduce

reduce方法是接收两个参数，第一个参数是函数，该函数作为累加器，数组中的每个值（从左到右）被传入该函数进行处理，第二个参数作为累加器的初始值, 如果未传入第二个参数，那第二个参数默认等于数组第一项。

  var arr = [1,2,3]
  console.log(arr.reduce((a, b) => a + b))//6
  console.log(arr.reduce((a, b) => a + b)， 5)//11
  console.log(arr)//[1,2,3]

# 结论：reduce方法返回根据参数函数进行处理的结果，并且不会影响原数组。

# some

some方法是接收一个函数作为参数，判断数组中是否存在满足函数条件的项，只要有一项满足条件，就会返回true。

  var arr = [1,2,3]
  console.log(arr.some(x => x > 2))//true
  console.log(arr)//[1,2,3]

# 结论：some方法返回根据参数判断的结果，只有数组中有一项满足条件就返回true，并且不会影响原数组。

# every

every方法是接收一个函数作为参数，判断数组中是否存在满足函数条件的项，只有数组每一项都满足条件，才会返回true。

  var arr = [1,2,3]
  console.log(arr.every(x => x > 2))//false
  console.log(arr.every(x => x > 0))//true
  console.log(arr)//[1,2,3]

# 结论：every方法返回根据参数判断的结果，只有数组中每一项都满足条件才返回true，并且不会影响原数组。

# filter

filter方法是接收一个函数作为参数，从数组中筛选满足函数条件的项，并将满足在的项组成一个新数组返回。

  var arr = [1,2,3]
  console.log(arr.filter(x => x > 1))//[2,3]
  console.log(arr)//[1,2,3]

# 结论：filter方法返回满足条件的项组成的新数组，并且不会影响原数组。

# map

map方法是接收一个函数作为参数，对数组中的每一项运行参数函数，返回每次函数调用的结果组成的数组。

  var arr = [1,2,3]
  console.log(arr.map(x => x * 2))//[2,4,6]
  console.log(arr)//[1,2,3]

# 结论：map方法返回每次参数函数调用的结果组成的数组，并且不会影响原数组。
